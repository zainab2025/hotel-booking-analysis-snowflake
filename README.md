# hotel-booking-analysis-snowflake
# Hotel Booking Project

This repository contains code exercises for a hotel booking system, created while following an online tutorial/course for learning purposes.  

## ⚡ Purpose
- Practice SQL and data manipulation.
- Learn how to work with Snowflake, CSV files, and data transformations.

## 🚀 Features
- Load CSV data into Snowflake tables.
- Basic data cleaning and formatting (e.g., INITCAP, TRIM, NULL handling).
- Query examples with CASE statements and formatting.

## ⚠️ Note
- These scripts are based on an educational tutorial and are **for learning purposes only**.
- They should **not be used for commercial purposes** without modification.
- Feel free to explore and modify the code to practice your skills.

## 📂 Files
- `BRONZE_HOTEL_BOOKING.sql` – SQL script to create and load the hotel booking table.
- `README.md` – This file.

## 💡 How to Use
1. Clone this repository.
2. Load the SQL scripts into your Snowflake environment.
3. Modify or extend the queries as needed for practice.
